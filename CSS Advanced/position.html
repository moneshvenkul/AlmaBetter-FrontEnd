Absolute Position

Absolute Parent (position: relative)
Absolute Box

Absolute boxes are removed from flow and positioned relative to the nearest positioned ancestor (this purple container).

Content after the absolute container continues as if the absolute box wasn't there.

Fixed Position

Look for the orange box in the top-right corner of the browser window. It stays there when you scroll the page.

Fixed Box

Fixed boxes are positioned relative to the viewport (browser window) and stay put when scrolling the main page.

Sticky Position

Scroll *inside* the teal box below. The header "Sticky Header" will stick to the top of the teal container when you scroll past it.

Sticky Header

This is the content inside the sticky container.

Keep scrolling down inside this teal box...

...until the header reaches the top of the container.

It should then "stick" to the top while you scroll the rest of this content.

Sticky positioning works relative to the nearest scrollable ancestor (or the viewport if none).
